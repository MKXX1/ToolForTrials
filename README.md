# ToolForTrials

Tool for see stats in trials(Time, Seed, Rank, Map, Ping, Fps)

Based on [Total](https://github.com/ShyKiss/Total)

# Bindings
| Key | Action |
|----------------|----------------|
| Del  |          Unhook |
| Home |          Rehook |
| Minus |         Hide Overlay |
